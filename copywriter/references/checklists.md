# Fast-Evaluation Checklists & Impact Order

This is the quick layer. Use it for rapid passes and to decide where to focus. When something here fails and you need the *why* or the fix, open the full reference file named in brackets.

---

## Impact Order (rank problems by this)

Copy elements are not equal. Fixing the top of this list moves response far more than the bottom. When you critique, lead with failures highest on this list — a broken headline outweighs a dozen word-choice nits.

1. **Audience / market match** — right person, right awareness stage [01-foundation.md]
2. **Offer** — what they get, price, risk reversal, bonuses [05-conversion.md]
3. **Headline** — the ad for the ad; 80% of the decision to keep reading [02-headlines-leads.md]
4. **Lead / opening** — enters the reader's existing mental conversation [02-headlines-leads.md]
5. **Proof** — claims without evidence are ignored [04-persuasion-proof.md]
6. **CTA** — specific action, benefit-framed, prominent [05-conversion.md]
7. **Body structure & flow** — every element earns the next [03-core-copy.md]
8. **Psychology & persuasion** — triggers, biases, trust [04-persuasion-proof.md]
9. **Writing craft** — clarity, concision, rhythm, voice [03-core-copy.md]
10. **Format fit** — medium-specific conventions [06-formats.md]

Note the tension: testing data (below) ranks *audience > offer > price > format > headline > body*. Both orderings agree that audience and offer dominate and that craft is lowest-leverage. Use impact order to sequence a critique; use testing order to sequence experiments.

## Headlines Checklist [02-headlines-leads.md]

- [ ] Contains a clear benefit
- [ ] Specific, not vague
- [ ] Flags the target audience
- [ ] Reader knows what's being sold in ~2 seconds
- [ ] Creates curiosity or urgency
- [ ] Matches the audience's awareness stage
- [ ] A competitor *couldn't* run the same headline (if they could, it's generic)
- [ ] Passes the "table test" (would you say it to someone across a table?)

## Leads / Openings Checklist [02-headlines-leads.md]

- [ ] Enters the conversation already happening in the reader's head
- [ ] Triggers emotion before logic
- [ ] Creates the "that's me" moment fast
- [ ] First sentence is short and compelling
- [ ] Each sentence earns the next
- [ ] Doesn't bore the reader with what they already know
- [ ] Problem is stated briefly (1-2 sentences), then moves on

## Body Copy Essentials [03-core-copy.md]

| Principle | Summary |
|-----------|---------|
| Slippery slide | Every element's job is to get the next element read |
| Opening paragraph | Short; hook immediately |
| Bucket brigades | "Here's why...", "But that's not all...", "Think about it..." |
| Cross-heads | Every 300-500 words; make some of them questions |
| Proof | Claims without evidence are ignored |
| Objection handling | Raise and resolve before the reader does |
| Fascinations | Curiosity bullets that tease without revealing |
| Long copy | Works when every word earns its place |

## Writing Craft Essentials [03-core-copy.md]

| Principle | Summary |
|-----------|---------|
| Clarity | Write to be understood, not to impress |
| Reading level | ~8th grade |
| Short words | "use" not "utilize" |
| Concrete > abstract | "23 pounds in 8 weeks" beats "significant weight loss" |
| Vary sentence length | Rhythm comes from variety |
| Short paragraphs | 1-4 sentences; use white space |
| Conversational voice | Write like you talk, but better |
| Omit needless words | Cut ~20% on revision |
| Read aloud | The ear catches what the eye misses |

## Offer Construction Checklist [05-conversion.md]

- [ ] Main offer clear and simple
- [ ] Bonuses add genuine value
- [ ] Guarantee prominent
- [ ] Price justified / anchored
- [ ] Reason given for any discount
- [ ] Trial/sample option if applicable
- [ ] Urgency element included

## CTA Checklist [05-conversion.md]

- [ ] Specific action verb
- [ ] First-person phrasing tested ("Start my free trial")
- [ ] States the benefit (what they GET)
- [ ] Visually prominent
- [ ] Multiple CTAs in long copy
- [ ] Supporting text handles the final objection

## Urgency / Scarcity Checklist [05-conversion.md]

- [ ] Specific deadline (date + time)
- [ ] Quantity limit with a real number
- [ ] Reason the limit exists
- [ ] Consequence of missing it
- [ ] Loss framing (what they miss out on)
- [ ] Scarcity is *real* and enforced

## Format Quick Tests [06-formats.md]

1. **Sales letters** — long enough to make the sale, strong P.S.?
2. **Print ads** — reads editorial, not "adsy"?
3. **Email** — single clear CTA, subject line tight?
4. **Landing pages** — one goal, form as short as possible?
5. **Video** — hook in first 5 seconds, brand ID early?
6. **Social ads** — thumb-stopping, platform-native?

## Testing Priority (experiment order) [07-testing.md]

1. Audience / list (highest leverage)
2. Offer
3. Price
4. Format
5. Headline
6. Body copy
7. Creative elements (lowest leverage)

## Benchmark Metrics (for grounding claims) [05-conversion.md, 07-testing.md]

| Lever | Reported effect | Source |
|-------|-----------------|--------|
| Personalized CTAs | ~202% better conversion | Conversion Code, Cashvertising |
| Live chat | ~2.8x conversion | Conversion Code |
| Emphasized CTAs | ~89% better | Conversion Code |
| Delay in responding | ~2x response lost | Ogilvy |
| Charm pricing ($39 vs $34) | $39 outsells | Cashvertising |

Treat these as directional, not guarantees — cite them as "one benchmark suggests," never as laws.

## Common-Mistakes Self-Audit [08-mistakes.md]

**Strategic** — creativity serves sales not awards? audience known specifically?
**Headlines** — clear (no puns/tricks)? promises a specific benefit?
**Copy** — benefits not features? specific not vague? enough proof? clear CTA?
**Structure** — clear hierarchy? offer prominent? right structure for the medium?
**Psychology** — matched to awareness stage? scarcity real? testimonials from the right people?
